# Design Tokens References

This skill is split into focused reference files so you can load only the token values that match the UI work. Tokens are centrally defined in `assets/scss/_variables.scss` and should not be changed without design approval.

## Route By Task

- Use [colors.md](colors.md) for palette, semantic, neutral, chip, and brand colors
- Use [typography-and-spacing.md](typography-and-spacing.md) for font family, font sizes, and spacing scale
- Use [breakpoints-and-figma.md](breakpoints-and-figma.md) for breakpoints, responsive mixin usage, and Figma-to-token mapping

## Cross-Skill Boundaries

- Use `nova-styling` for SCSS mixins and layout patterns
- Use `figma-to-angular` when the task starts from a mockup

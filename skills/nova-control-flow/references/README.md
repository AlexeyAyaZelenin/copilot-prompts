# Nova Control Flow References

This skill is split into focused reference files so you can load only the control-flow guidance that matches the template work. These references target Angular v17 built-in control flow as used in Nova.

## Route By Task

- Use [if-and-switch.md](if-and-switch.md) for `@if`, `@else`, `@else if`, and `@switch`
- Use [for-and-tracking.md](for-and-tracking.md) for `@for`, contextual variables, `@empty`, and `track` guidance
- Use [migration-and-review.md](migration-and-review.md) for structural-directive migration, behavioral differences, pitfalls, and review questions

## Cross-Skill Boundaries

- Use `nova-angular-standards` for broader template conventions
- Use `nova-signals` when the template logic should move into computed or signal-driven state
